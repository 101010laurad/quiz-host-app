## Installation

-   Run `install-dev.sh`
-   Go to **Setup &gt; Remote Site Settings** and add the Heroku site URL.
-   Go to **Setup &gt; Custom Metadata Types** and add a **Quiz Settings** record.

If you want to build and contribute, run `npm install`.
